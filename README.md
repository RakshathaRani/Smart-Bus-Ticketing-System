> **BCA Final Year Project (Completed: August 2024)**

A web-based **Smart Bus Ticketing & Reservation System** developed using **Python, Django, MySQL, HTML, CSS, JavaScript, and Bootstrap**.

The application provides a complete online bus booking experience, allowing users to search buses, reserve seats, simulate payments, generate PDF tickets, and manage bookings through a user-friendly interface. An administrative dashboard is also included for managing buses, routes, cities, seats, users, and bookings.

> **Note:** The source code for this project is intentionally kept private. This repository is created to showcase the project features, workflow, and user interface.

---

# 📖 Project Overview

The objective of this project was to develop an efficient online bus reservation system that reduces manual booking processes and improves the overall user experience.

The system allows users to:

- Create an account
- Login securely
- Search available buses
- View bus details
- Select seats
- Make simulated payments
- Download PDF tickets
- View booking history
- Cancel bookings (within the allowed cancellation period)

Administrators can manage all project data using the Django Admin Dashboard.
